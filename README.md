# Copy-Files-FlexCache
Copy files from one directory to another using FlexCache technology

################################################

This code will copy files from SRC directory to DST direcotry
using Flexcache.

It loop each file under directory and search for file.flag,
if it exist it will copy the 2 files to DST direcotry, and delete the file.flag
if not, it will pass to the next file
